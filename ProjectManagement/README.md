# Sprint Belgeleri

Bu klasör, akademinin Scrum template yapısına uygun olarak sprint kanıtlarını barındırır.

- `Sprint1Documents/`, `Sprint2Documents/`, `Sprint3Documents/` — her sprintin board screenshotları, daily scrum notları (jpeg/docx), ürün ekran görüntüleri.
- `assets/` — logo ve genel görseller.

Dosyalar eklendikçe README.md ilgili sprint bölümünden linklenir.
